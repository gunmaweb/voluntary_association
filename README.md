## Gunma.web の皆様へ
- 自分がGunma.web というコミュニティを作り 4年が立ちました。
そろそろ自分の役割を他の人にもやって貰いたいと思い、
「任意団体」に移行するという決断をした次第です。  
  
とはいえこれですぐに「見捨てる」的な事はしませんし...  
イキナリ初参加の人に自分の役割を押し付けるような事をする  
つもりもありません。  
  
出来れば今までよりも「Gunma.web」というコミュニティを支えて  
下さる皆さんに運営に関わっていただければ..と思った次第です。  
  
2014/10/4 金澤 宏昭
